Movies Api MyFlix

This Api will provide the users with information about their favorite movies. They can sign up, log in and save their favorite movies.

  • 1. Get data of all movies

  • Description: Return data of all movies

  • URL: http://localhost:8080/movies

  • Method Type: GET

  • Query Parameters: none

  • Request Format: N/A

  • Response Format: json

  • 2. Get data of a movie

  • Description: Return data of a movies

  • URL: http://localhost:8080/movies/:title

  • Method Type: GET

  • Query Parameters: Title of the movie

  • Request Format: N/A

  • Response Format: json

  • 3. Get data of movies by genre

  • Description: Return data of movies by genre

  • URL: http://localhost:8080/movies/genre/:genre

  • Method Type: GET

  • Query Parameters: Genre name

  • Request Format: N/A

  • Response Format: json

  • 4. Get data of movies by director

  • Description: Return data of movies by director

  • URL: http://localhost:8080/movies/director/:director

  • Method Type: GET

  • Query Parameters: Director name

  • Request Format: N/A

  • Response Format: json

  • 5. Register a user

  • Description: Register a user

  • URL: http://localhost:8080/users

  • Method Type: POST

  • Query Parameters: none

  • Request Format: json - Example - {name: 'Barrack Obama', username: 'bobama123', password: 'xxx', dob: '01/01/2000'}

  • Response Format: json

  • 6. Update user info

  • Description: Update information of user

  • URL: http://localhost:8080/users/update/:username

  • Method Type: PUT

  • Query Parameters: username

  • Request Format: json - Example - {name: 'Barrack Obama', username: 'bobama123', password: 'xxx', dob: '01/01/2000'}

  • Response Format: json

  • 7. Add movie as favorite

  • Description: Add a specific movie as favorite

  • URL: http://localhost:8080/users/addfav/:username/:title

  • Method Type: PUT

  • Query Parameters: Title, username

  • Request Format: N/A

  • Response Format: json

  • 8. Remove movie as favorite

  • Description: Remove a movie as favorite

  • URL: http://localhost:8080/users/remfav/:username/:title

  • Method Type: PUT

  • Query Parameters: Title, username

  • Request Format: N/A

  • Response Format: json

  • 9. Deregister user

  • Description: Deregister a user

  • URL: http://localhost:8080//users/:username

  • Method Type: DELETE

  • Query Parameters: username

  • Request Format: N/A

  • Response Format: json
